# OpenRouter Setup

Set `AI_PROVIDER=openrouter`, configure `OPENROUTER_API_KEY`, and choose model IDs for `AI_FAST_MODEL`, `AI_REASONING_MODEL`, `AI_VISION_MODEL`, and `AI_FALLBACK_MODEL`.

Optional headers `OPENROUTER_SITE_URL` and `OPENROUTER_APP_NAME` are applied only inside the adapter. Privacy routing defaults deny data collection and require supported parameters for structured outputs and tools.
